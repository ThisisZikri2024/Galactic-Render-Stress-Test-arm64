# 🌌 Omniverse Engine v3.3 [ARM64 Edition]
> **Extreme-Performance Browser Benchmark & Celestial Simulation Optimized for ARM64 Architecture.**

Omniverse Engine v3.3 adalah alat penanda aras (benchmark) yang dibina khusus untuk menguji had pemprosesan grafik pada peranti seni bina **ARM64** (Apple M-Series, Snapdragon, MediaTek). Berbeza dengan versi standard, edisi ini menggunakan **GPU-Accelerated Canvas Rendering** untuk memaksimumkan throughput data tanpa membebankan CPU.

## 🚀 Ciri-Ciri Utama (ARM64 Optimized)

* **⚡ Unlocked Overdrive:** Menghapuskan had 60fps/120fps. Menyokong paparan 144Hz hingga 240Hz secara asli.
* **🎨 GPU Canvas Core:** Menggunakan `Canvas 2D Context` dengan `{ alpha: false }` untuk akses terus ke perkakasan grafik ARM64.
* **✨ 40s Persistence:** Partikel RGB kekal selama 40 saat untuk menguji kecekapan *Memory Management* dan *Garbage Collection*.
* **📱 Multi-Touch Engine:** Dioptimumkan untuk input skrin sentuh berbilang titik (iPad Pro/Android Flagship).
* **🪐 Orbital Physics:** Simulasi matematik trigonometri ($\sin$, $\cos$) untuk pergerakan planet dan 300+ asteroid secara masa nyata.
